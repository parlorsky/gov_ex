# Задача 1. Равновесная цена: функции спроса и предложения.

## Условие
Функции спроса (\(D\)) и предложения (\(S\)) имеют вид:
\[D = 2p'' - 2p' + 20,\]
\[S = 2p'' + 2p' + 3p + 8.\]

Найти зависимость равновесной цены \(p = p(t)\) от времени, если в начальный момент \(p = 4,\ p' = 2\).

## Решение

### Идея
В динамической модели Уолраса цена реагирует на избыточный спрос. С условиями \(p(0)=4\) и \(p'(0)=2\) задача имеет смысл как **уравнение второго порядка**, где «ускорение» цены равно избыточному спросу:
\[p'' = D - S.\]
Подставляя:
\[p'' = (2p'' - 2p' + 20) - (2p'' + 2p' + 3p + 8) = -4p' - 3p + 12,\]
откуда
\[\boxed{p'' + 4p' + 3p = 12.}\]

(Если читать \(D = S\) как чисто алгебраическое равенство, члены \(2p''\) сократились бы и осталось бы 1-го порядка \(4p' + 3p = 12\) — но тогда условие \(p'(0)=2\) было бы лишним и противоречило бы уравнению.)

### Решение ОДУ
Линейное неоднородное с постоянными коэффициентами.

**Однородное:** \(\lambda^2 + 4\lambda + 3 = 0 \Rightarrow (\lambda+1)(\lambda+3) = 0 \Rightarrow \lambda_1 = -1,\ \lambda_2 = -3\).
\[p_h = C_1 e^{-t} + C_2 e^{-3t}.\]

**Частное:** ищем \(p_p = A\) (правая часть — константа). Подставляя: \(3A = 12 \Rightarrow A = 4\).

**Общее:** \(p(t) = 4 + C_1 e^{-t} + C_2 e^{-3t}\).

### Подбор констант
\(p'(t) = -C_1 e^{-t} - 3 C_2 e^{-3t}\).

\[\begin{cases} p(0) = 4 + C_1 + C_2 = 4, \\ p'(0) = -C_1 - 3 C_2 = 2. \end{cases}\Rightarrow\ C_1 + C_2 = 0,\ -C_1 - 3 C_2 = 2.\]

Из первого \(C_1 = -C_2\); подставляя: \(C_2 - 3C_2 = -2 C_2 = 2 \Rightarrow C_2 = -1\), \(C_1 = 1\).

### Ответ
\[\boxed{p(t) = 4 + e^{-t} - e^{-3t}.}\]

Проверка: \(p(0) = 4 + 1 - 1 = 4\) ✓, \(p'(0) = -1 + 3 = 2\) ✓.

### Поведение траектории
- \(p(0) = 4\), монотонно растёт до пика, затем плавно возвращается к равновесному уровню \(p_\infty = 4\).
- Численная проверка показывает максимум вблизи \(t \approx 0{,}5\) (\(p \approx 4{,}38\)), затем экспоненциальное затухание.
- Оба собственных значения отрицательные ⇒ устойчивая «передемпфированная» система: колебаний нет.
