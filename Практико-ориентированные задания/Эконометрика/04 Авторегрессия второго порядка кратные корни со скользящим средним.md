# Задача 4. Авторегрессия второго порядка с кратными корнями со скользящим средним.

## Условие
Заданы следующие параметры модели авторегрессии второго порядка с кратными корнями характеристических уравнений: $N = 3000$, $M(x) = 0$, $D(x) = 5$, $D(\varepsilon) = 0{,}25$.

Представьте программный код для генерации такого случайного процесса авторегрессии скользящего среднего в окне $M = 3$ и построения графика на экране на языке программирования по вашему выбору.

## Решение

### Программа минимум

```python
import numpy as np
import matplotlib.pyplot as plt

N = 3000
mean_x = 0.0
var_x = 5.0
var_eps = 0.25
window = 3

def variance_equation(u):
    # u = r^2, а отношение var_x / var_eps равно 20.
    return (1 + u) / ((1 - u) ** 3) - var_x / var_eps

# Подбираем u простым делением отрезка пополам.
left = 0.01
right = 0.99
for step in range(60):
    middle = (left + right) / 2
    if variance_equation(middle) > 0:
        right = middle
    else:
        left = middle

u = (left + right) / 2
r = np.sqrt(u)
phi1 = 2 * r
phi2 = -u
constant = mean_x * (1 - phi1 - phi2)

sigma_eps = np.sqrt(var_eps)
eps = np.random.normal(0, sigma_eps, N)

x = np.zeros(N)
for t in range(2, N):
    x[t] = constant + phi1 * x[t - 1] + phi2 * x[t - 2] + eps[t]

# Скользящее среднее с окном 3.
ma = []
for start in range(0, N - window + 1):
    window_values = x[start:start + window]
    ma.append(window_values.mean())
ma = np.array(ma)

plt.plot(x, alpha=0.5)
plt.plot(range(window - 1, N), ma)
plt.show()
```

### Полное решение

### Идея
AR(2) с кратными корнями (как в задаче 2: $\varphi_1 = 2\lambda$, $\varphi_2 = -\lambda^2$). В этой задаче $r$ не задан — параметр $\lambda$ выбираем из условия совместимости заданных $D(x)=5$ и $D(\varepsilon)=0{,}25$ через связь:
$$\sigma_x^2 = \frac{(1+\lambda^2) \sigma_\varepsilon^2}{(1-\lambda^2)^3}.$$
Решая численно $\frac{1+u}{(1-u)^3} = 20$ ($u = \lambda^2$): $u \approx 0{,}572$, $\lambda \approx 0{,}756$.

### Код

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.optimize import brentq

# подбор λ из условия совместимости D(x)=5 и D(eps)=0.25
f = lambda u: (1+u)/(1-u)**3 - 20
u = brentq(f, 0.01, 0.99)
lam = np.sqrt(u)
phi1, phi2 = 2*lam, -lam**2

N, M, D_e, win = 3000, 0.0, 0.25, 3
c = M * (1 - phi1 - phi2)
sigma_e = np.sqrt(D_e)

rng = np.random.default_rng(13)
x = np.zeros(N)
e = rng.normal(0, sigma_e, N)
for t in range(2, N):
    x[t] = c + phi1*x[t-1] + phi2*x[t-2] + e[t]
x = x[200:]   # burn-in

ma = np.convolve(x, np.ones(win)/win, mode="valid")

plt.figure(figsize=(10, 3))
plt.plot(x, alpha=0.5, lw=0.5, label="x_t")
plt.plot(np.arange(win-1, len(x)), ma, lw=0.9, label=f"MA({win})")
plt.legend(); plt.tight_layout(); plt.show()
```

### Прогон
```
λ=0.7561, φ1=1.5122, φ2=-0.5717
x:       mean=0.061, var=4.45      (цель D=5)
MA(3):   mean=0.064, var=4.23
```

### Замечания
- Расхождение $\mathrm{var}\approx 4{,}45$ vs цели $5$ объясняется конечной длиной выборки и переходным процессом; на нескольких реализациях средняя дисперсия сойдётся к $5$.
- Скользящее среднее $M = 3$ слабее задавливает дисперсию, чем $M=5$, и меньше деформирует низкочастотные компоненты.
