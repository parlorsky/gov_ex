# Задача 3. Решающее дерево для классификации точек на плоскости.

## Условие
Предложить вариант решающего дерева для классификации объектов, предложенных на рисунке.

Рисунок («Vectors to be Classified») — двумерная диаграмма рассеяния в осях $P(1)$ и $P(2)$. Два класса объектов:
- класс «o» (кружки): сосредоточен в области $P(1) \in [-2;\ 2]$, $P(2) \in [-2;\ 1]$;
- класс «+» (плюсы): сосредоточен в области $P(1) \in [3;\ 7]$, $P(2) \in [3;\ 7]$.

Сопроводить дерево графической интерпретацией решения на плоскости рисунка.

## Решение

### Программа минимум

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.tree import DecisionTreeClassifier, plot_tree

# Простые учебные точки: два кластера из условия.
X = np.array([
    [-2, -1], [-1, 0], [0, -2], [1, 1], [2, 0],
    [3, 3], [4, 5], [5, 4], [6, 7], [7, 6],
])
y = np.array(["o", "o", "o", "o", "o", "+", "+", "+", "+", "+"])

# Дерево глубины 1 ищет один понятный вопрос: например, P(2) <= 2.
tree = DecisionTreeClassifier(max_depth=1, random_state=0)
tree.fit(X, y)

plt.scatter(X[:, 0], X[:, 1], c=(y == "+"))
plt.axhline(2, color="black")
plt.xlabel("P(1)")
plt.ylabel("P(2)")
plt.show()

plot_tree(tree, feature_names=["P(1)", "P(2)"], class_names=tree.classes_)
plt.show()
```

### Идея
Кластеры разделены и по $P(1)$ и по $P(2)$ — годится одно осепараллельное разбиение по любой оси. Возьмём $P(2) \approx 2$ как разделяющую горизонталь:
- $P(2) \le 2$ — кластер «o» (нижний);
- $P(2) > 2$ — кластер «+» (верхний).

Альтернатива — $P(1) \approx 2{,}5$, аналогично.

### Дерево

```
              [ корень ]
              P(2) ≤ 2 ?
              /        \
            да           нет
             ↓            ↓
          класс 'o'    класс '+'
```

### Графическая интерпретация на плоскости

```
P(2)
 ↑
 7 │            + + + + +
 6 │             +  +  + +
 5 │            + + + + +
 4 │              + + +
 3 │                 +
─2─┼────────────────────────  ← P(2) = 2  (разделяющая прямая)
 1 │  o  o
 0 │  o o o o o
-1 │   o  o o
-2 │   o
   └─────────────────────→  P(1)
     -2 -1 0 1 2 3 4 5 6 7
```

Точки выше горизонтали $P(2) = 2$ — класс «+», ниже — класс «o». Поскольку оба кластера лежат строго по разные стороны этой линии, ошибок на обучающей выборке ноль.

### Замечания
- Глубины 1 (один тест) достаточно — данные **линейно разделимы** (даже параллельно осям).
- Для **робастности** к новым точкам можно выбрать прямую как середину «зазора»: $P(2) = (1 + 3)/2 = 2$.
- Эквивалентно подходит $P(1) \le 2{,}5$ или диагональное разбиение $P(1) + P(2) \le 4$ — все классифицируют тренировочную выборку без ошибок, но имеют разные обобщающие свойства. Чем больше зазор от точек до границы — тем устойчивее правило.
- В sklearn это `DecisionTreeClassifier(max_depth=1)`; дерево само выбрало бы похожий порог по информативному критерию (Gini / entropy).
