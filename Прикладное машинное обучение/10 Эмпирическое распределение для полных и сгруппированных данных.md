# Эмпирическое распределение для полных и сгруппированных данных. Огива и гистограмма. Оценка Нельсона-Алена кумулятивной функции доли риска.

## TL;DR
**Эмпирическая функция распределения (ЭФР)** $F_n(x) = (1/n)\#\{i: X_i\le x\}$ — несмещённая, состоятельная оценка $F$. По теореме Гливенко–Кантелли $\sup|F_n - F|\to 0$ почти наверное. Для **сгруппированных данных** строят **гистограмму** (плотность по интервалам) и **огиву** (полигон накопленных частот, кусочно-линейная аппроксимация $F$). При наличии цензурирования — **оценка Нельсона–Алена** кумулятивной интенсивности $\hat\Lambda(t) = \sum_{t_i\le t} d_i/n_i$ и связанная **оценка Каплана–Мейера** функции выживания $\hat S(t) = \prod_{t_i\le t}(1 - d_i/n_i)$.

## Развёрнуто

### Полные данные
Выборка $X_1,\dots,X_n$ — независимые одинаково распределённые из $F$.

**Эмпирическая функция распределения**:
$$F_n(x) = \frac{1}{n}\sum_{i=1}^n \mathbb 1\{X_i\le x\}.$$
Свойства:
- $F_n(x)$ — несмещённая оценка $F(x)$;
- $\mathrm{Var}\,F_n(x) = F(x)(1-F(x))/n$;
- **Гливенко–Кантелли**: $\sup_x|F_n(x)-F(x)|\xrightarrow{п.н.} 0$ при $n\to\infty$;
- $\sqrt n(F_n - F)$ слабо сходится к мосту Брауна — основа критериев Колмогорова–Смирнова.

**Эмпирические моменты**: $\bar X = \sum X_i/n$, $S^2 = \sum(X_i - \bar X)^2/(n-1)$.

### Сгруппированные данные
Данные разбиты на интервалы $[a_0, a_1), [a_1, a_2), \dots, [a_{m-1}, a_m]$ с частотами $n_j$, $\sum n_j = n$.

**Гистограмма** — оценка плотности:
$$\hat f(x) = \frac{n_j}{n\cdot(a_j - a_{j-1})}\quad \text{при }x\in[a_{j-1}, a_j).$$
Площадь под гистограммой = 1.

**Огива** — кусочно-линейная аппроксимация $F$. Соединяем точки $(a_j, F_n(a_j))$ отрезками. Удобно для получения $F$ на промежуточных значениях, особенно если внутри интервала плотность считается равномерной.

Предположение равномерности внутри интервала даёт также формулы для оценок моментов:
$$\bar X\approx \frac{1}{n}\sum_j n_j\cdot \tilde a_j,\quad \tilde a_j = (a_{j-1}+a_j)/2.$$

### Цензурированные данные и функция выживания
В страховании, медицине, надёжности — есть **цензурирование** (правое): для части наблюдений известно лишь, что $X_i > c_i$.

Обозначения для каждого момента наблюдаемых событий $t_1 < t_2 < \dots$:
- $n_i$ — число объектов «под риском» непосредственно перед $t_i$ (живых, не выбывших, не цензурированных);
- $d_i$ — число событий (исков, смертей) в момент $t_i$;
- $c_i$ — число цензурирований в $[t_i, t_{i+1})$.

**Функция выживания** $S(t) = P(X > t) = 1 - F(t)$. **Кумулятивная функция доли риска (cumulative hazard)** $\Lambda(t) = -\ln S(t) = \int_0^t h(s)\,ds$, $h$ — функция доли риска (hazard rate).

### Оценка Нельсона–Алена
$$\hat\Lambda(t) = \sum_{t_i\le t}\frac{d_i}{n_i}.$$
Это естественная аппроксимация $\int dN/Y$, где $N$ — счётчик событий, $Y$ — число под риском.

**Свойства**:
- асимптотически несмещённая;
- дисперсия (оценка):
$$\widehat{\mathrm{Var}}\hat\Lambda(t) = \sum_{t_i\le t}\frac{d_i}{n_i^2}.$$

**Оценка функции выживания через Нельсона–Алена**:
$$\hat S_{NA}(t) = \exp(-\hat\Lambda(t)).$$

### Оценка Каплана–Мейера (для сравнения)
$$\hat S_{KM}(t) = \prod_{t_i\le t}\left(1 - \frac{d_i}{n_i}\right).$$
Идея: на каждом наблюдаемом моменте события — условная вероятность пережить домножается на $(1 - d_i/n_i)$.

При отсутствии цензуры $\hat S_{KM}(t) = 1 - F_n(t)$.

Для $d_i$ малых $n_i$ оценки Нельсона–Алена и Каплана–Мейера близки: $\ln(1 - d_i/n_i)\approx -d_i/n_i$. На малых выборках Нельсона–Алена обычно более устойчива (не уходит в 0 при $d_i = n_i$).

### Пример
Данные о времени до отказа: 5 объектов, события в моменты $t = 2, 5, 7$. В каждый момент — одно событие, цензурирования нет; ниже $n_i = 5, 4, 3$, $d_i=1$.
$$\hat\Lambda(2) = 1/5 = 0{,}2,\ \hat\Lambda(5) = 0{,}2 + 1/4 = 0{,}45,\ \hat\Lambda(7) = 0{,}45 + 1/3 = 0{,}783.$$
$\hat S_{NA}(7) = e^{-0{,}783}\approx 0{,}457$.
$\hat S_{KM}(7) = 0{,}8\cdot 0{,}75\cdot 0{,}667 = 0{,}4$.

### Применение
- Описательная статистика.
- Тестирование гипотез о распределении (КС, $\chi^2$).
- Бутстреп — основан на ЭФР.
- В страховании — оценка функций выживания клиентов, портфеля, моментов аннулирования.
- В надёжности — оценка интенсивности отказов.

### Подводные камни
- Гистограмма чувствительна к ширине интервала — слишком мелкие интервалы шумят, слишком крупные сглаживают детали.
- Огива предполагает равномерность внутри интервала — может вводить смещение в моменты.
- Оценка Нельсона–Алена и Каплана–Мейера предполагают **независимость** цензурирования от события.
- При сильном цензурировании в хвосте оценки очень шумные — возможны искажения.
