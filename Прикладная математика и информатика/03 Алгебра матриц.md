# Алгебра матриц. Линейные преобразования. Собственные значения и собственные векторы.

## TL;DR
Матрица $m\times n$ — таблица чисел. Операции: $+$, $\cdot \alpha$, умножение $A\cdot B$ (когда столбцов $A$ = строк $B$), транспонирование. Каждое линейное преобразование $\varphi: V \to W$ в выбранных базисах задаётся матрицей; смена базиса — преобразование подобия $A' = P^{-1}AP$. **Собственный вектор**: $Av = \lambda v$, $v\ne 0$; **собственное значение** $\lambda$ — корень характеристического многочлена $\det(A-\lambda E)=0$. Если у $n\times n$ матрицы $n$ ЛНЗ собственных векторов — она диагонализуема: $A = P\Lambda P^{-1}$.

## Развёрнуто

### Матрицы и операции
$A = (a_{ij})_{m\times n}$. Операции:
- сложение и умножение на скаляр — поэлементно (для одинаковых размеров);
- умножение $C = AB$, $c_{ij} = \sum_k a_{ik}b_{kj}$, размер $m\times p$ при $A$ $m\times n$, $B$ $n\times p$. Не коммутативно ($AB\ne BA$ в общем), ассоциативно, дистрибутивно;
- транспонирование $(A^\top)_{ij} = a_{ji}$, $(AB)^\top = B^\top A^\top$.

### Определитель и обратная
$\det A$ для $n\times n$: Лейбниц $\det A = \sum_\sigma \mathrm{sgn}(\sigma)\prod a_{i,\sigma(i)}$ или разложение по строке/столбцу. Свойства: $\det(AB)=\det A\det B$, $\det A^\top=\det A$, при перестановке строк меняет знак.

$A$ обратима $\iff \det A\ne 0$. Формула: $A^{-1} = \mathrm{adj}(A)/\det A$, где $\mathrm{adj}(A)_{ij} = (-1)^{i+j}M_{ji}$ (присоединённая, минор $M_{ji}$).

Методы вычисления: Гаусс-Жордан (приписать $E$ справа и привести к ($E\,|\,A^{-1}$)), через присоединённую — для теории и малых размеров.

### Ранг
Ранг $r(A)$ — максимальный порядок ненулевого минора $=$ число ЛНЗ строк $=$ число ЛНЗ столбцов. Сохраняется при элементарных преобразованиях. Связан с системой $Ax=b$: совместна $\iff r(A)=r(A|b)$ (Кронекер–Капелли); число свободных переменных $= n - r$.

### Линейные преобразования
$\varphi: V \to W$ называется линейным, если $\varphi(\alpha u + \beta v) = \alpha\varphi(u) + \beta\varphi(v)$. В базисах $e_1,\dots,e_n$ ($V$) и $f_1,\dots,f_m$ ($W$) преобразование задаётся матрицей $A$: столбцы $A$ — координаты $\varphi(e_j)$ в базисе $\{f_i\}$. Композиция операторов = умножение матриц.

**Замена базиса** в $V$ через матрицу перехода $P$ ($v = Pv'$): матрица оператора $\varphi: V\to V$ преобразуется как $A' = P^{-1}AP$ (преобразование подобия). Подобные матрицы имеют одинаковый характеристический многочлен, ранг, след, определитель.

Ядро $\ker\varphi = \{v : \varphi v = 0\}$, образ $\mathrm{Im}\,\varphi$. Теорема о ранге: $\dim\ker\varphi + \dim\mathrm{Im}\,\varphi = \dim V$.

### Собственные значения и векторы
Для $A: V\to V$ ($n\times n$):
$$Av = \lambda v, \quad v\ne 0.$$
$\lambda$ — собственное значение, $v$ — собственный вектор. Условие нетривиальности: $\det(A - \lambda E) = 0$ — **характеристическое уравнение**, корни — спектр $A$.

Алгебраическая кратность $\lambda$ — её кратность в характеристическом многочлене. Геометрическая — $\dim\ker(A-\lambda E)$. Геометрическая $\le$ алгебраической.

**Свойства**:
- $\sum \lambda_i = \mathrm{tr}\,A$, $\prod \lambda_i = \det A$.
- Собственные векторы для разных $\lambda$ ЛНЗ.
- Симметричные вещественные матрицы имеют только вещественные $\lambda$ и ортогональный базис из собственных векторов (спектральная теорема).

### Диагонализация
$A$ диагонализуема, если существует базис из её собственных векторов. Тогда $A = P\Lambda P^{-1}$, $\Lambda = \mathrm{diag}(\lambda_1,\dots,\lambda_n)$, столбцы $P$ — собственные векторы. Условие: алгебраическая = геометрической для каждого $\lambda$. Не диагонализуемые матрицы приводятся к жордановой нормальной форме.

### Пример
$A = \begin{pmatrix}2 & 1 \\ 1 & 2\end{pmatrix}$. $\det(A-\lambda E) = (2-\lambda)^2 - 1 = 0 \Rightarrow \lambda_1=1,\lambda_2=3$. Собственные векторы: $v_1=(1,-1)^\top$, $v_2=(1,1)^\top$. $P = \begin{pmatrix}1&1\\-1&1\end{pmatrix}$, $A = P\,\mathrm{diag}(1,3)\,P^{-1}$.

### Подводные камни
- Не каждая квадратная матрица диагонализуема (пример: $\begin{pmatrix}1&1\\0&1\end{pmatrix}$ — кратное $\lambda=1$ с одной собственной прямой).
- Собственные значения вещественной матрицы могут быть комплексными.
