# Линейные дифференциальные уравнения первого порядка и методы их решения. Задача Коши. Существование и единственность решения задачи Коши.

## TL;DR
- **Линейные дифференциальные уравнения первого порядка и методы их решения:** уравнение имеет вид $y'+p(x)y=q(x)$; однородное решение равно $y_o=C\exp\left(-\int p(x)\,dx\right)$.
- **Методы решения:** неоднородное уравнение решают [[#Метод интегрирующего множителя|интегрирующим множителем]], [[#Метод Бернулли (подстановка $y = u\cdot v$)|подстановкой $y=u\cdot v$]] или [[#Метод вариации постоянной (Лагранж)|вариацией постоянной]]; все методы дают $y=e^{-P(x)}\left(\int q(x)e^{P(x)}\,dx+C\right)$.
- **Задача Коши:** это линейное ОДУ с начальным условием $y(x_0)=y_0$; константа в общем решении выбирается из этого условия.
- **Существование и единственность решения задачи Коши:** при непрерывных $p$ и $q$ линейная задача Коши имеет единственное решение на всём интервале их непрерывности; общий критерий даёт [[#Теорема существования и единственности|теорема Пикара–Линделёфа]].

## Развёрнуто

### Постановка
Линейное ОДУ первого порядка:
$$y' + p(x)\,y = q(x), \quad x\in I.$$
$p, q$ — заданные непрерывные функции. **Однородное** уравнение: $q(x)\equiv 0$.

### Однородное уравнение
$y' = -p(x)y$ — с разделяющимися переменными:
$$\frac{dy}{y} = -p(x)\,dx \Rightarrow y = C\exp\left(-\int p(x)\,dx\right).$$

### Метод интегрирующего множителя
Умножим обе части на $\mu(x) = \exp(\int p(x)\,dx)$:
$$\mu y' + \mu p\,y = \mu q \Leftrightarrow (\mu y)' = \mu q.$$
Интегрируя:
$$\mu(x)\,y = \int \mu(x)\,q(x)\,dx + C \Rightarrow y = \frac{1}{\mu(x)}\left(\int \mu(x)\,q(x)\,dx + C\right).$$

### Метод Бернулли (подстановка $y = u\cdot v$)
Подстановка: $y' = u'v + uv'$. Уравнение:
$$u'v + uv' + p\,uv = q \Leftrightarrow u'v + u(v' + pv) = q.$$
Выберем $v$ так, чтобы $v' + pv = 0$ — однородное, $v = \exp(-\int p\,dx)$. Тогда $u'v = q$, $u = \int q/v\,dx + C$. $y = uv$.

### Метод вариации постоянной (Лагранж)
Сначала находим общее решение однородного $y_o = C\exp(-\int p\,dx)$. Затем «варьируем» $C\to C(x)$: ищем $y = C(x)\exp(-\int p\,dx)$. Подстановка в исходное даёт
$$C'(x) = q(x)\exp\left(\int p\,dx\right),$$
откуда $C(x) = \int q\exp(\int p\,dx)\,dx + C_0$.

### Структура решения
Общее решение неоднородного: $y = y_o + y_{\text{ч}}$, где $y_o$ — общее решение однородного (с константой), $y_{\text{ч}}$ — частное решение неоднородного. Все три метода дают эквивалентный результат:
$$y(x) = e^{-P(x)}\left(\int q(x)\,e^{P(x)}\,dx + C\right), \quad P(x) = \int p(x)\,dx.$$

### Задача Коши
$$y' + p(x)y = q(x), \quad y(x_0) = y_0.$$
Для линейного ОДУ задача Коши имеет единственное явное решение:
$$y(x) = e^{-P(x)+P(x_0)}\left[y_0 + \int_{x_0}^x q(s)\,e^{P(s)-P(x_0)}\,ds\right],$$
где $P(x)=\int_{x_0}^x p(t)\,dt$.

### Теорема существования и единственности
Для общего ОДУ $y' = f(x,y)$, $y(x_0) = y_0$ — **теорема Пикара–Линделёфа**: если $f$ непрерывна по $(x,y)$ в окрестности $(x_0,y_0)$ и удовлетворяет условию Липшица по $y$ (например, $\partial f/\partial y$ ограничена), то решение существует и единственно в некоторой окрестности $x_0$.

Для линейного ОДУ $f(x,y) = -p(x)y + q(x)$ — Липшиц по $y$ автоматический при непрерывной $p$. Более того, решение **продолжается** на весь интервал непрерывности $p, q$.

Доказательство — методом последовательных приближений Пикара: $y_{k+1}(x) = y_0 + \int_{x_0}^x f(s, y_k(s))\,ds$, сходимость по теореме Банаха о неподвижной точке (см. вопрос 05).

### Пример
Решить $y' - 2y/x = x^3$, $y(1) = 3$, $x>0$.

**Метод интегрирующего множителя.** $p = -2/x$, $\mu = \exp(-2\int dx/x) = e^{-2\ln x} = 1/x^2$.
$$(y/x^2)' = x^3/x^2 = x \Rightarrow y/x^2 = x^2/2 + C \Rightarrow y = x^4/2 + Cx^2.$$
Условие $y(1)=3$: $1/2 + C = 3 \Rightarrow C = 5/2$. Ответ: $y = x^4/2 + 5x^2/2$.

### Подводные камни
- Если $p$ или $q$ имеют особенность на интервале, решение может не существовать сквозь неё.
- Уравнение Бернулли $y' + py = qy^n$, $n\ne 0,1$ — линейным **не является**, но сводится к линейному заменой $z = y^{1-n}$.
- Не путать «метод Бернулли» (как метод решения линейного через $y=uv$) с «уравнением Бернулли» (нелинейное обобщение).
