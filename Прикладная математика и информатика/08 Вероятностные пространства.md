# Вероятностные пространства. Независимость событий. Формула полной вероятности. Формула Байеса.

## TL;DR
- **Вероятностные пространства:** это тройка $(\Omega,\mathcal F,P)$, где $\Omega$ — элементарные исходы, $\mathcal F$ — [$\sigma$-алгебра событий](#вероятностное-пространство-колмогоров-1933), а $P$ — вероятностная мера с аксиомами Колмогорова.
- **Независимость событий:** события $A$ и $B$ независимы, если $P(A\cap B)=P(A)P(B)$; при $P(B)>0$ это равносильно $P(A|B)=P(A)$.
- **Формула полной вероятности:** для полной группы гипотез $\{H_i\}$ вероятность события равна $P(A)=\sum_i P(H_i)P(A|H_i)$.
- **Формула Байеса:** по наблюдению $A$ пересчитывает априорную вероятность гипотезы в апостериорную: $P(H_k|A)=\frac{P(H_k)P(A|H_k)}{P(A)}$.

## Развёрнуто

### Вероятностное пространство (Колмогоров, 1933)
- $\Omega$ — пространство **элементарных исходов**.
- $\mathcal F\subseteq 2^\Omega$ — **$\sigma$-алгебра**: содержит $\Omega$, замкнута относительно дополнения и счётных объединений. Элементы $\mathcal F$ — события.
- $P: \mathcal F\to[0,1]$ — **вероятностная мера**:
   1. $P(A)\ge 0$;
   2. $P(\Omega)=1$;
   3. для попарно непересекающихся $A_1,A_2,\dots$: $P(\bigsqcup A_i) = \sum P(A_i)$ (счётная аддитивность).

### Свойства вероятности
- $P(\varnothing)=0$, $P(\bar A)=1-P(A)$;
- монотонность: $A\subseteq B\Rightarrow P(A)\le P(B)$;
- $P(A\cup B) = P(A)+P(B)-P(A\cap B)$ (формула включений-исключений);
- непрерывность: $A_n\uparrow A\Rightarrow P(A_n)\to P(A)$.

### Типичные модели
- **Классическая** (конечное равновероятное $\Omega$): $P(A) = |A|/|\Omega|$.
- **Геометрическая**: $P(A) = \mu(A)/\mu(\Omega)$ (мера Лебега).
- **Дискретная**: $\Omega = \{\omega_1,\omega_2,\dots\}$, $P(\omega_i)=p_i\ge 0$, $\sum p_i=1$.

### Условная вероятность
Для $P(B)>0$:
$$P(A|B) = \frac{P(A\cap B)}{P(B)}.$$
$P(\cdot|B)$ — снова вероятностная мера на $\mathcal F$.

**Формула умножения**: $P(A_1\cap\dots\cap A_n) = P(A_1)P(A_2|A_1)\cdots P(A_n|A_1\cap\dots\cap A_{n-1})$.

### Независимость событий
$A$ и $B$ **независимы**:
$$P(A\cap B) = P(A)P(B).$$
Эквивалентно (при $P(B)>0$): $P(A|B)=P(A)$.

**Совместная независимость** $\{A_i\}$: для любого конечного подмножества индексов $P(\bigcap A_{i_k}) = \prod P(A_{i_k})$. Сильнее, чем попарная (классический контрпример — независимы попарно, но не в совокупности).

### Формула полной вероятности
$\{H_1, H_2,\dots\}$ — **полная группа** (разбиение): $\bigsqcup H_i = \Omega$, $H_i\cap H_j=\varnothing$ при $i\ne j$, $P(H_i)>0$. Тогда для любого $A\in\mathcal F$:
$$P(A) = \sum_i P(H_i)\,P(A|H_i).$$

### Формула Байеса
В условиях формулы полной вероятности и $P(A)>0$:
$$P(H_k|A) = \frac{P(H_k)\,P(A|H_k)}{\sum_i P(H_i)\,P(A|H_i)} = \frac{P(H_k)\,P(A|H_k)}{P(A)}.$$
$P(H_k)$ — априорная вероятность гипотезы; $P(H_k|A)$ — апостериорная. Используется в байесовской статистике, ML (наивный байесовский классификатор), медицинской диагностике.

### Пример
Тест на болезнь: $P(+|\text{больной})=0{,}99$, $P(+|\text{здоровый})=0{,}05$, распространённость $P(\text{больной})=0{,}01$.
$$P(+) = 0{,}01\cdot 0{,}99 + 0{,}99\cdot 0{,}05 = 0{,}0099 + 0{,}0495 = 0{,}0594.$$
$$P(\text{больной}|+) = 0{,}0099 / 0{,}0594 \approx 0{,}167.$$
Хотя тест точный, при низкой априорной вероятности апостериорная всё равно низка — иллюстрация важности базовой ставки.

### Подводные камни
- Условная вероятность не симметрична: $P(A|B)\ne P(B|A)$ (ошибка прокурора).
- Попарная независимость не влечёт независимость в совокупности.
- $\sigma$-аддитивность принципиальна: только конечной аддитивности недостаточно для теории.
- В геометрической схеме «парадокс Бертрана» показывает, что нужно фиксировать механизм случайности.
