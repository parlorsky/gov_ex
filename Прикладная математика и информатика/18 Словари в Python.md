# Словари в Python. Назначение, особенности, создание, изменение. Основные операции и методы. Способы итерации по словарям.

`dict` — встроенный отображающий тип Python, реализован как **хеш-таблица с открытой адресацией** (псевдослучайное зондирование с perturbation) поверх **компактного представления**: массив индексов + плотный массив записей в порядке вставки (с 3.6+). Хранит пары `ключ-значение`; ключи должны быть **хешируемыми** — требуются корректные `__hash__` и `__eq__` (равные объекты обязаны иметь равные хеши). Доступ, вставка, удаление — O(1) в среднем, O(n) в худшем случае (массовые коллизии). **Коллизии** разрешаются открытой адресацией (альтернатива — цепочки, как в Java HashMap); удаление помечает ячейку tombstone. При load factor ~2/3 — ресайзинг с перехешированием. Хеш строк рандомизирован между запусками (защита от HashDoS). Создание: `{}`, `dict(...)`, dict comprehension. Основные методы: `get`, `setdefault`, `update`, `pop`, `keys/values/items`. С Python 3.7+ сохраняется порядок вставки. Итерация — по ключам по умолчанию.
