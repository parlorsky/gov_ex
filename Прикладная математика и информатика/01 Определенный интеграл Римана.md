# Определенный интеграл Римана. Свойства определенного интеграла. Применение определенного интеграла в экономических моделях.

Определённый интеграл Римана $\int_a^b f(x),dx$ — предел интегральных сумм $\sigma(T,\xi) = \sum f(\xi_i)\Delta x_i$ при $\lambda(T) = \max \Delta x_i \to 0$, не зависящий от разбиения и выбора точек $\xi_i$. Существует для непрерывных и кусочно-непрерывных функций (критерий Лебега: ограничена + точки разрыва имеют меру ноль). Не интегрируемы: функция Дирихле, $1/x$ на $[-1,1]$.

**Свойства:**
- Линейность: $\int(\alpha f + \beta g) = \alpha\int f + \beta\int g$.
- Аддитивность: $\int_a^b = \int_a^c + \int_c^b$.
- Монотонность: $f \le g \Rightarrow \int f \le \int g$; следствие $|\int f| \le \int |f|$.
- Теорема о среднем: для непрерывной $f$ существует $c \in [a,b]$ с $\int_a^b f,dx = f(c)(b-a)$ — площадь под графиком равна площади одного прямоугольника со средней высотой.

**Считается через формулу Ньютона-Лейбница:** $\int_a^b f,dx = F(b) - F(a)$, где $F' = f$. Доказательство через $\Phi(x) = \int_a^x f,dt$ и теорему Барроу ($\Phi' = f$ по теореме о среднем + непрерывность).

**В экономике:**

- Потребительский излишек: $CS = \int_0^{q^*} D(q)\,dq - p^* q^*$.
- Дисконтированный поток платежей: $PV = \int_0^T f(t)e^{-rt},dt$; бессрочная рента $PV = c/r$.
- Накопление от потоковой величины: $\Delta P = \int_{t_1}^{t_2} P'(t),dt$ (поток → запас).
