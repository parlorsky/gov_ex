# Определенный интеграл Римана. Свойства определенного интеграла. Применение определенного интеграла в экономических моделях.

## TL;DR
Определённый интеграл Римана $\int_a^b f(x)\,dx$ — предел интегральных сумм $\sum f(\xi_i)\Delta x_i$ при измельчении разбиения. Существует для непрерывных и кусочно-непрерывных функций. Свойства: линейность, аддитивность по отрезку, монотонность, теорема о среднем. Считается через формулу Ньютона-Лейбница: $\int_a^b f\,dx = F(b) - F(a)$. В экономике — потребительский излишек, накопленная прибыль, дисконтированная стоимость непрерывного потока платежей.

## Развёрнуто

### Определение
Пусть $f: [a,b] \to \mathbb{R}$ ограничена. Разбиение $T = \{a = x_0 < x_1 < \dots < x_n = b\}$, диаметр $\lambda(T) = \max \Delta x_i$, где $\Delta x_i = x_i - x_{i-1}$. На каждом отрезке выбираем точку $\xi_i \in [x_{i-1}, x_i]$ — получаем размеченное разбиение. **Интегральная сумма Римана**:
$$\sigma(T, \xi) = \sum_{i=1}^n f(\xi_i)\,\Delta x_i.$$
Функция $f$ **интегрируема по Риману** на $[a,b]$, если существует конечный предел
$$I = \lim_{\lambda(T)\to 0} \sigma(T, \xi),$$
не зависящий ни от разбиения, ни от выбора точек $\xi_i$. Этот предел и называется $\int_a^b f(x)\,dx$.

### Суммы Дарбу и критерий интегрируемости
$m_i = \inf_{[x_{i-1},x_i]} f$, $M_i = \sup f$. Нижняя и верхняя суммы Дарбу:
$$s(T) = \sum m_i \Delta x_i, \qquad S(T) = \sum M_i \Delta x_i.$$
**Критерий Дарбу**: $f$ интегрируема $\iff \lim_{\lambda(T)\to 0}(S(T) - s(T)) = 0$, т.е. $\sum \omega_i \Delta x_i \to 0$, где $\omega_i = M_i - m_i$ — колебание.

**Критерий Лебега**: $f$ интегрируема по Риману $\iff$ $f$ ограничена и множество её точек разрыва имеет лебегову меру нуль.

Достаточные условия интегрируемости:
- $f$ непрерывна на $[a,b]$;
- $f$ монотонна на $[a,b]$;
- $f$ ограничена и имеет конечное (или счётное с предельной точкой) число разрывов первого рода.

### Свойства
1. **Линейность**: $\int_a^b (\alpha f + \beta g)\,dx = \alpha\int_a^b f\,dx + \beta\int_a^b g\,dx$.
2. **Аддитивность по отрезку**: $\int_a^b f\,dx = \int_a^c f\,dx + \int_c^b f\,dx$ для любого $c$.
3. **Монотонность**: если $f \le g$, то $\int_a^b f \le \int_a^b g$. Следствие: $\left|\int_a^b f\,dx\right| \le \int_a^b |f|\,dx$.
4. **Оценка**: $m(b-a) \le \int_a^b f\,dx \le M(b-a)$, где $m \le f \le M$.
5. **Теорема о среднем**: для непрерывной $f$ существует $c \in [a,b]$ такая, что $\int_a^b f\,dx = f(c)(b-a)$.
6. $\int_a^a f = 0$, $\int_a^b f = -\int_b^a f$.

### Формула Ньютона-Лейбница
Если $f$ непрерывна на $[a,b]$ и $F$ — её первообразная ($F'=f$), то
$$\int_a^b f(x)\,dx = F(b) - F(a).$$
Связана с теоремой Барроу: функция $\Phi(x) = \int_a^x f(t)\,dt$ дифференцируема, $\Phi'(x) = f(x)$.

Стандартные приёмы вычисления: замена переменной $\int_a^b f(\varphi(t))\varphi'(t)\,dt = \int_{\varphi(a)}^{\varphi(b)} f(u)\,du$, интегрирование по частям $\int_a^b u\,dv = uv\Big|_a^b - \int_a^b v\,du$.

### Геометрический смысл
Для $f \ge 0$ — площадь криволинейной трапеции под графиком; в общем случае — алгебраическая площадь со знаком.

### Применение в экономических моделях

**1. Потребительский излишек (consumer surplus).** Если $p = D(q)$ — функция спроса, $q^*$ — равновесный объём, $p^*$ — равновесная цена, то
$$CS = \int_0^{q^*} D(q)\,dq - p^* q^*.$$
Геометрически — площадь между кривой спроса и горизонталью $p = p^*$. Аналогично производительский излишек $PS = p^* q^* - \int_0^{q^*} S(q)\,dq$.

**2. Дисконтированная стоимость непрерывного потока платежей.** Если поток дохода — $f(t)$ ден.ед./год, ставка непрерывного дисконтирования $r$, то приведённая стоимость потока за горизонт $T$:
$$PV = \int_0^T f(t)\,e^{-rt}\,dt.$$
Для постоянного потока $f(t) \equiv c$: $PV = \frac{c}{r}(1 - e^{-rT})$. Бессрочная рента ($T \to \infty$): $PV = c/r$.

**3. Накопленная прибыль / выпуск.** Если $P'(t)$ — мгновенная скорость прибыли, то прибыль за период $[t_1, t_2]$ равна $\int_{t_1}^{t_2} P'(t)\,dt$.

### Подводные камни
- Интеграл Римана определён только на отрезке для ограниченной функции — несобственные интегралы (бесконечный промежуток или неограниченность) — отдельная теория.
- Функция Дирихле (1 на рациональных, 0 на иррациональных) не интегрируема по Риману, но интегрируема по Лебегу.
- Формула Ньютона-Лейбница требует существования первообразной; для разрывной $f$ нужна осторожность.
